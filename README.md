# Cipe Violência
Aplicativo desenvolvido com  Ionic + Angular, para auxiliar estudantes de enfermagem a diagnosticar abusos com crianças e ajudar com as escolhas de tratamento efetivo.

# Tecnologias
* Ionic 4
* Angular 7
